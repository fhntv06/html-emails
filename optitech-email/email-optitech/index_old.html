НАЧАЛО ПРОЕКТА ПО МОДЕРНИЗАЦИИ ОАО «ГОЛАЗ»

В апреле начат масштабный проект по реконструкции ОАО «ГолАЗ». Проект ведет рабочая группа ООО «ОПТИТЭК АГРО», которая состоит из следующих сотрудников:

Дмитрий Сурнин, Ирина Долгополиченская, Дмитрий Артюшин. Руководит рабочей группой Вячеслав Васенков.

Необходимую поддержку также оказывают сотрудники различных подразделений.

В ходе осуществления проекта будут реконструированы несколько корпусов заводcкого здания (частично или полностью), которые в дальнейшем будут использоваться для склада запасных частей и производства (крупноузловой сборки) сельскохозяйственных машин тех брендов, которые входят в портфолио ООО «ОПТИТЭК АГРО».

Также планируется организация помещений для тренинг-центра: оборудованые классы для теоретических занятий и мастерские для практических занятий.

На данный момент проектной группой был проведён тендер для выбора подрядной организации для разработки проекта, выбрана подрядная организация и начаты работы по разработке проектной документации.

В ближайшее время начнутся работы по обследованию зданий и примыкающей территории.

На основании результатов обследования будет разработана проектная документация, далее будут выбраны подрядчики для проведения строительных работ.
